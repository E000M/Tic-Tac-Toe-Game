## Tic Tac Toe - Tkinter Game

A simple and playful Tic Tac Toe game made with Python Tkinter, designed with fun colors and simple animations.  
The game includes 2-Player mode, Player-vs-Computer mode, score tracking, a restart button, and a main menu.



## Game Features
-  **PvP Mode** - Two players take turns.
-  **PvC Mode** - Play against a simple computer AI.
-  **Simple UI** - Colors, rounded buttons, playful fonts.
-  **Restart Button** - Instantly reset the board.
-  **Main Menu** - Switch between modes.
-  **Scoreboard** - Automatically tracks wins for X and O, or Tie.

## The AI Behavior in the Game
The computer opponent:
- Tries to win if possible.
- Blocks the player if they are about to win.
- Otherwise picks a random empty spot.
